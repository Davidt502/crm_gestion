📚 APIs disponibles
Usa el header X-API-Token: TU_TOKEN en cada petición. Haz clic en un endpoint para ver más detalles.
GET /api/pub/clientes Lista clientes activos

Devuelve hasta 100 clientes con estado Activo.

curl https://crm-gestion.onrender.com/api/pub/clientes \
     -H "X-API-Token: TU_TOKEN"

// Respuesta

{ "clientes": [{ "id_cliente": 1, "nombre": "Juan Pérez", "correo": "juan@mail.com", "estado": "Activo" }], "total": 1 }
GET /api/pub/proveedores Lista proveedores activos

Devuelve hasta 100 proveedores con su categoría.

curl https://crm-gestion.onrender.com/api/pub/proveedores \
     -H "X-API-Token: TU_TOKEN"

// Respuesta

{ "proveedores": [{ "id_proveedor": 1, "nombre_empresa": "Tech S.A.", "nit": "123456-7", "estado": "Activo", "nombre_categoria": "Tecnología" }], "total": 1 }
GET /api/pub/empleados Lista empleados activos

Devuelve hasta 100 empleados con su dependencia.

curl https://crm-gestion.onrender.com/api/pub/empleados \
     -H "X-API-Token: TU_TOKEN"

// Respuesta

{ "empleados": [{ "id_empleado": 1, "nombre": "María López", "cargo": "Analista", "estado": "Activo", "nombre_dependencia": "Sistemas" }], "total": 1 }
